My server

How I set up my server, & what I use it for

How I got my server and specs

I'll just touch on this briefly, but I got my server for free from the co-owner of Pi-Architects (Architectural firm), Mr. Warrick. The server itself is a Dell Poweredge T630, and the specs are as follows:

Dual Intel Xeon E5-2620 v3 (24 cores in total)
64 gb DDR4 ram
24 tb raw storage, 20x600 gb 10k rpm hard drives, 2x600 gb 15k hard drives (raid 1 boot), 8x1.2 tb 10k hard drives

OS and initial setup

The first thing I had to do when I set up my server was wipe the drives, as I was contractually obligated to do so. After like 8 hours, the drives were wiped, and I could move on to installing a distro. Before I installed a distro, I had to pick one. This part was tricky, since there were a lot of options, and I'll list a few of them here:

  • Alpine
  • Debian 11 stable
  • Ubuntu Server
  • Arch
I ended up settling on Debian 11 stable, mostly due to me knowing how to use Debian, and not really wanting to deal with removing Snap from Ubuntu Server.
The installation/initial setup process was pretty painless, the hardest thing that I had to do was setting up ZFS with 28 drives (ended up ripping out 20 of them due to the immense power draw).
